# Investment Growth Simulator

An interactive web application that demonstrates how compound interest and consistent monthly investing can contribute to long-term wealth growth.

## About the Project

The Investment Growth Simulator allows users to experiment with different investment scenarios by changing:

- Initial investment
- Monthly contribution
- Expected annual return
- Investment time period

The application calculates the estimated future portfolio value and visually compares total contributions with investment growth.

## Features

- Interactive investment calculator
- Compound growth simulation
- Monthly contribution calculations
- Dynamic portfolio growth chart
- Comparison between total contributions and investment growth
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- GitHub
- Vercel

## How It Works

The simulator applies monthly investment growth based on the user's selected annual return rate and monthly contribution.

Users can change the inputs to explore how different investment strategies may affect long-term portfolio growth.
